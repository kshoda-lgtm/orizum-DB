## 📋 主な特徴

### バイト向け
- **カスタムGPTの活用:** 定型フォーマットで高速・高精度なデータ入力
- **スプシへの直貼り:** GPT出力をそのままペーストするだけ

### 営業向け
- **4つのビュー:** 「今日やるべきこと」が一目瞭然
- **追い漏れゼロ:** システムが自動で次のアクション日を管理
- **AIによるDM提案:** 複数の文案から選んで送信（Phase 2）

### システム向け
- **自動更新:** 毎日深夜にYouTube指標を自動更新
- **エラー可視化:** 失敗した行とその理由を記録
- **スケーラビリティ:** 1000行規模から数万行規模への拡張に対応

## ✅ 受入条件

### Phase 0 完了時点
- バイトがカスタムGPTの出力をスプシに追加できる
- 営業が「未DM」「追い対象」ビューで日次業務を完結できる
- 追い漏れが構造的に発生しない

### Phase 1 完了時点
- YouTube登録者数が毎日自動で更新される
- 更新失敗時の理由が可視化される

### Phase 2 完了時点
- Webアプリで複数条件での検索が可能
- クリエイター詳細画面で全情報が1画面に集約される
